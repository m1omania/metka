# Metka - USDT Web App Prototype

Прототип веб-приложения для операций с цифровым долларом (USDT). Минималистичный, интуитивный и безопасный интерфейс для глобального рынка.

## Технологии

- React 18
- TypeScript
- Vite
- React Router
- QR Code генерация

## Структура проекта

```
src/
  ├── navigation/       # Навигация приложения
  ├── screens/          # Экраны приложения
  ├── components/       # Переиспользуемые компоненты
  ├── services/         # Сервисы и mock-данные
  ├── utils/            # Утилиты
  ├── styles/           # Тема и стили
  └── types/            # TypeScript типы
```

## Основные функции

### 1. Экран Баланс
- Крупное отображение баланса USDT
- Цифровая клавиатура для ввода суммы
- Кнопки: "Запросить", "QR", "Получить"
- Просмотр и копирование адреса кошелька
- История транзакций
- Заблокированные опции пополнения/вывода

### 2. Экран Отправить/Получить
- Два таба: "Запросить" и "Отправить"
- Генерация ссылок: `https://metka.one/r/xxxx` (request) и `https://metka.one/s/xxxx` (send)
- Генерация QR-кодов
- История транзакций (входящие/исходящие)

### 3. Экран Настройки
- Личный кабинет (email)
- Информация о валютах и статусе бэкапа
- Настройки безопасности (FaceID/TouchID)
- Инструкции по безопасности
- Поддержка (Telegram)

## Установка и запуск

```bash
# Установка зависимостей
npm install

# Запуск dev-сервера
npm run dev

# Сборка для production
npm run build

# Просмотр production сборки
npm run preview
```

Приложение откроется автоматически в браузере на `http://localhost:3000`

## Особенности прототипа

- **Mock-данные**: Все данные хранятся локально через localStorage
- **Минималистичный дизайн**: Крупная типографика, много "воздуха", легкая визуальная иерархия
- **Адаптивный дизайн**: Работает на всех размерах экранов
- **Безопасность**: Информация о non-custodial кошельке и хранении ключей

## Компоненты

- `BalanceDisplay` - Отображение баланса
- `DigitalKeyboard` - Цифровая клавиатура
- `TransactionHistory` - История транзакций
- `QRCodeModal` - Модальное окно с QR-кодом
- `LinkShareModal` - Модальное окно со ссылкой
- `WalletAddress` - Отображение адреса кошелька
- `EmptyState` - Пустое состояние

## Mock-данные

Приложение использует mock-данные для демонстрации функциональности:
- Баланс: 1250.50 USDT
- Промо-баланс: $50
